## ❓ 프로젝트 소개

- 전국에 있는 뮤지컬에 대한 정보를 모아놓은 서비스입니다.
- 뮤지컬에 대한 정보 제공
- 비로그인식 즐겨찾기 기능

<br>

## 💻 개발 환경

- 기술 스택 : Next.js, Typescript, React, Styled-component
- 버전 관리 : GitHub
- 툴 : Photoshop
- 배포 : AWS
